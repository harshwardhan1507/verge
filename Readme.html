01 · About

What is MemoryOS?

Our brains are poor at organizing lived experiences. MemoryOS acts as an external brain — a structured, searchable, and visually rich interface for everything that matters to you.

Memories are not just "notes." They are typed and color-coded across seven meaningful dimensions: commitments you've made, people in your life, emotions you've felt, patterns you've noticed, events that shaped you, things left unresolved, and insights you've gained.

Built in under 48 hours, MemoryOS combines a high-performance React frontend with a lightweight Node.js/SQLite backend — easy to self-host with zero external database dependencies.

02 · Features

What's Inside

🧠

Typed Memory System

Seven semantic memory types, each with its own visual identity and color system.

💾

Zero-config Persistence

All data stored in a local SQLite database via better-sqlite3 — no cloud required.

✨

Smooth Animations

Fluid transitions and micro-interactions powered by GSAP 3.

🌐

WebGL Canvas

Immersive visual background rendered with OGL, the lightweight WebGL library.

⚡

React Compiler

Automatic memoization eliminates unnecessary renders without manual optimization.

🎨

Dark-first Design

Custom Tailwind theme with a deep dark base and vibrant, semantic accent colors.

🔒

Accessible Components

Built on Radix UI primitives — dialogs, tooltips, scroll areas, and more.

🗺️

Client-side Routing

Fast SPA navigation with React Router v7, no page reloads.

📦

Monorepo Structure

Express API and Vite frontend co-exist and run concurrently from one repo.

04 · Setup

Getting Started

1

Clone the repository

bash 
git clone https://github.com/harshwardhan1507/Hackathon-verge.git
cd Hackathon-verge
2

Install dependencies

bash 
npm install
3

Start the full stack (recommended)

Runs both the Express API server and the Vite dev server simultaneously.

bash 
npm run dev:full

# Frontend → http://localhost:5173
# Backend  → http://localhost:3000
4

Build for production

bash 
npm run build    # Type-check + bundle
npm run preview  # Preview production build

06 · Memory Types

The Seven Memory Types

MemoryOS uses a semantic color system to visually distinguish categories of memory at a glance. Each type has a dedicated color token in the Tailwind config.

Commitment #f59e0b

Promises made to yourself or others that need to be honored.

Person #0ea5e9

Memories tied to significant people in your life.

Emotion #f43f5e

Felt experiences and emotional states worth remembering.

Pattern #a855f7

Recurring themes, habits, and behavioral loops you've noticed.

Event #22c55e

Significant moments and milestones that shaped your path.

Unresolved #f97316

Open threads and loose ends that are still on your mind.

Insight #14b8a6

Realizations, learnings, and moments of clarity.
